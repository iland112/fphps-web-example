Welcome to the FastPass JAVA SDK Demo!

This application demonstrates the capabilities of the FastPass e-Passport Reader SDK in a web environment.

Connected Device: DEVICE-SERIAL-NOT-FOUND

How to Use This Application

Passport (e-Passport)

Reads ICAO-9303 compliant electronic passports.
- **Manual:** Place the passport and click to read immediately.
- **Auto:** Automatically detects and reads the passport when placed on the device.

ID Card

Reads various national electronic ID cards. The operation is similar to passport reading.

Barcode

Reads 1D and 2D barcodes, such as those found on boarding passes or visas.

Scan Page

Captures raw, full-page images from the scanner under different light sources (White Light, Infrared, Ultraviolet).

Device Settings

Allows you to configure the device's operational parameters, such as which features are enabled (e-Passport, ID Card, Barcode) and other advanced options.
